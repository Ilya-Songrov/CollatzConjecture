### Collatz conjecture
It is also known as the 3n + 1 problem, the 3n + 1 conjecture, the Ulam conjecture.

![Screen Shot](md/maxresdefault.jpg)

## How to build/run
```
# Build
git clone <this_repo>
cd <this_repo>
mkdir -p build/tmp
cd build/tmp
cmake ../..
make

# Run
cd ../bin/Release_Linux_x86_64/
./CollatzConjecture --input_num 123123
```
