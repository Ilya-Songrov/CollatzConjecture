### Collatz conjecture
It is also known as the 3n + 1 problem, the 3n + 1 conjecture, the Ulam conjecture.

## How to build
```
git clone <this_repo>
cd <this_repo>
mkdir -p build/tmp
cd build/tmp
cmake ../..
```
